# Éditer un billet et citer avec Zotero — Guide complet

Bienvenue dans l’éditeur d’Athanor. Ce billet “showcase” vous guide pas à pas pour :
- écrire en Markdown,
- créer des liens internes (backlinks),
- citer avec Zotero (Better BibTeX),
- afficher automatiquement la bibliographie.

> Note : vous pouvez coller tel quel un billet généré par un LLM. S’il contient un front matter YAML ou un titre `#`, le système déduira automatiquement le titre et le slug.

## 1) Découvrir l’éditeur (Markdown)

- Gras : `**important**` → **important**  
- Italique : `*nuancé*` → *nuancé*  
- Titres : `#`, `##`, `###`  
- Listes :
  1. Première idée
  2. Seconde idée
- Lien externe : `[Athanor](https://athanor-philosophy-platform.vercel.app/)`

Exemple minimal collable (avec front matter, **non compilé** ici) :
```md
---
title: "Mon super billet"
tags: ["demo", "markdown"]
excerpt: "Une petite démonstration"
---

# Mon super billet

Du texte avec **gras**, *italique*, et un [lien](https://exemple.org).

Sans front matter, un simple # Mon super titre au début suffit : le titre et le slug seront inférés.

2) Mailler vos idées (backlinks)

Forme wiki (recommandée) :

[[slug-du-billet]]
[[slug-du-billet|Alias lisible]]

Forme HTML (si besoin) — affichée ici comme code pour éviter tout rendu :

<a href="/billets/slug-du-billet" class="backlink" data-backlink="slug-du-billet">Alias lisible</a>

Astuce : utilisez le bouton “Backlink” pour chercher et insérer.

3) Citer avec Zotero (Better BibTeX)

Pré-requis (côté auteur) :
	•	Installer Zotero + l’extension Better BibTeX (BBT).
	•	Ajouter vos références dans le groupe Zotero d’Athanor.
	•	Chaque référence possède une clé BBT (ex. Adorno1984, Boulnois1994, Brandom2019).

Insérer une citation via l’éditeur : cliquer “Citer”, chercher par auteur/titre/année/clé, insérer.
Ou bien saisir directement (ici non compilé) :

<Cite item="Adorno1984" />

Exemples en contexte (toujours montrés en code pour ne pas générer d’indice) :

La notion de « réconciliation extorquée » <Cite item="Adorno1984" /> éclaire le propos.
Voir aussi <Cite item="Boulnois1994" /> et une lecture moderne de Hegel <Cite item="brandomSpiritTrustReading2019" />.
Hors-série Lukács (Romanesques) <Cite item="RomanesquesLukacs2024" />.

Le validateur signale les clés inconnues et propose des corrections proches. L’éditeur affiche aussi la clé BBT pour vérification.

4) Afficher la bibliographie

Placer ce composant en fin de billet (affiché ici en code, donc non exécuté) :

<Bibliography />

Parcourir toutes nos références :

<a href="/billets/bibliographie-collective" class="backlink" data-backlink="bibliographie-collective">Index bibliographique</a>

6) Raccourcis utiles
	•	**gras**, *italique*, # Titre, > Citation, 1.  (liste numérotée)
	•	Backlinks : [[slug]] ou [[slug|Alias]]
	•	Citations : `&lt;Cite item="CleBBT" /&gt;` puis fin de billet : `<Bibliography />`
	•	Blocs de code : séquence de fences (montrée sans exécuter) :

```md
# Un exemple Markdown
Citez ainsi : <Cite item="Adorno1984" />
```

7) Bonnes pratiques
	•	Citer tôt et souvent : traçabilité, vérifiabilité, respect des sources.
	•	Préférer des clés BBT claires (AuteurAnnée).
	•	Maillage interne généreux : le graphe d’Athanor s’enrichit avec chaque backlink.

8) Résumé express
	1.	Écrivez en Markdown, 2) reliez avec [[slug]] (ou le bouton “Backlink”), 3) citez avec <Cite item="Block1978" />, 4) terminez par <Bibliography />.
C’est tout — bon billet.
